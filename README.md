## ✨ 功能特性

### 🏠 首页
- **品牌视觉** - 莫兰迪色系，低饱和高级感设计
- **加载动画** - 优雅的进度条与文字轮播过渡
- **极简文案** - 干净利落的品牌表达

### 💬 对话页面
- **双状态切换** - 空白欢迎态 ↔ 对话消息态平滑过渡
- **消息收发** - 完整的消息发送与AI自动回复功能
- **局部滚动** - 聊天区独立滚动，侧边栏与导航固定
- **+号菜单** - 图片/文件发送入口（UI框架）
- **侧边栏** - 角色状态、回应模式、今日动态展示

### 🎨 设计规范
- **配色系统** - 与首页100%统一的莫兰迪色系
- **图标系统** - 统一的20×20px线性图标
- **动效规范** - 克制高级的过渡动画（0.2s-0.4s ease-out）
- **响应式** - 桌面/平板/移动端全适配

## 🚀 快速开始

### 本地运行

```bash
# 克隆仓库
git clone https://github.com/feiting2hao/linger-web.git

# 进入目录
cd linger-web

# 启动本地服务器（任选一种）
python3 -m http.server 8080
# 或
npx serve .
```

然后访问 `http://localhost:8080`

## 🎯 设计原则

1. **品牌统一性** - 全站配色、字体、图标100%统一
2. **信息极简主义** - 文案「能短不长，能无则无」
3. **克制无装饰** - 无渐变、无炫彩、无多余动效
4. **层级优先** - 核心信息前置，视觉权重清晰
